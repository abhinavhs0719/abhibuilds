The person behind the builds

Not a prodigy.
Just someone building anyway.

I didn't start with a roadmap. I started with a decision — that I'd rather build something imperfect and learn from it than wait until I felt "ready." That decision changed everything.

Web development wasn't love at first sight. It was more like stubborn curiosity. I broke things, fixed them wrong, broke them again — and somewhere in that cycle, I realized the building itself was the point. Not the destination.

Now I'm focused on building strong web development fundamentals. I'm currently learning Node.js, building projects, and documenting the journey publicly through YouTube, LinkedIn, and this website.

The journey so far

The Start

Decided to stop consuming and start creating. Wrote my first HTML file. It was ugly. I was proud of it.

The Grind

The Shift
